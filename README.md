# PDF to JPG High Resolution Converter

A premium, high-quality PDF to JPG conversion tool with a sophisticated, gallery-like design.

## Features

- **High Quality Conversion**: Support for up to 1200 DPI with adjustable JPEG quality
- **Client-Side Processing**: All conversions happen in the browser, no server uploads required
- **Multi-Page Support**: Process all pages in PDF documents simultaneously
- **Parallel Processing**: Optimized conversion with adjustable parallel processing
- **Flexible Downloads**: Options for individual JPGs or organized ZIP packages
- **Modern Design**: Premium, gallery-like experience with sophisticated animations

## Design Highlights

- **Premium Color Scheme**: Deep black background with gold and teal accents
- **Typography**: Elegant Inter font with custom features and thoughtful spacing
- **Animation System**: Smooth, natural animations with custom keyframes
- **Glass Morphism**: Modern glass effects for UI components
- **Responsive Design**: Mobile-first approach with adaptive layouts

## Technology Stack

- Next.js for the React framework
- Tailwind CSS for styling
- Framer Motion for advanced animations
- PDF.js for PDF rendering
- Canvas API for image processing
- JSZip for download packaging

## Running Locally

First, install the dependencies:

```bash
npm install
# or
yarn install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build for Production

```bash
npm run build
# or
yarn build
```

Then start the production server:

```bash
npm start
# or
yarn start
```
